Aroma Yogini

Aroma Yogini is a paid membership site that offers access to products and content centered on yoga and meditation. Based on a member’s subscription level, members have products such as essential oils mailed to them on a monthly basis and get access to videos, audio, and written posts.

One prominent page on the Aroma Yogini site is called Library which gives access to video, audio, and written content in an organized tabbable panes layout via Bootstrap. To be able to update this content, we set up admin tools to allow the owner of Aroma Yogini to add, update, and delete content as the owner choses to do so. With there being different types of content on this page for members to access, the admin tools that we built needed to be able to distinguish where the content they were adding would display. Using form submissions, we are able to trigger the create, update, or delete functions in the controller.

With the create function, we pass in the data to the controller that is generated by what the admin fills out using their admin controls. The function first checks to see if the admin has included image and/or audio files in their post creation. It then goes and assigns the remaining input values to their corresponding columns in the MySQL table we have connected to the controller.

Similarly to the create function, the update function passes in the data from what the admin choses to update. However, in order for the controller to know which row we want to update in the table, we also pass in the id of said column when the admin submits their update request. The id is used with Laravel’s findorfail method to locate the row we want to change. Another noticeable difference with the update function verses the create function is that instead of utilizing the create method, we simply note which columns we want to update and then call the save function to apply the changes to be made.

Laravel’s destroy function allows for an easy way to remove rows from the table we created. All that is needed to pass in the id of the row we want to delete is use the findorfail method to locate the row, and then call the delete function which will delete the row from the table.
